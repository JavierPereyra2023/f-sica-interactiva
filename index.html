Desplazamiento

El desplazamiento es un vector que describe el cambio en la posición de un objeto. Es la distancia en línea recta desde el punto inicial hasta el punto final, junto con la dirección.

Δx = xf - xi

Donde:

  • Δx = Desplazamiento
  • xf = Posición final
  • xi = Posición inicial

Velocidad Media

La velocidad media es la relación entre el desplazamiento total y el tiempo total transcurrido. Es un vector que indica qué tan rápido y en qué dirección se ha movido un objeto desde su posición inicial hasta su posición final.

vmedia = Δx / Δt

Donde:

  • vmedia = Velocidad media
  • Δx = Desplazamiento
  • Δt = Intervalo de tiempo

Aceleración Media

La aceleración media es la relación entre el cambio en la velocidad y el tiempo transcurrido. Describe qué tan rápido cambia la velocidad de un objeto, ya sea en magnitud o dirección.

amedia = Δv / Δt

Donde:

  • amedia = Aceleración media
  • Δv = Cambio en la velocidad
  • Δt = Intervalo de tiempo
