Desplazamiento

El desplazamiento es un vector que describe el cambio en la posición de un objeto.

Δx = xf - xi

Donde: Δx = Desplazamiento, xf = Posición final, xi = Posición inicial

Velocidad Media

La velocidad media es la relación entre el desplazamiento total y el tiempo total transcurrido.

vmedia = Δx / Δt

Donde: vmedia = Velocidad media, Δx = Desplazamiento, Δt = Intervalo de tiempo

Aceleración Media

La aceleración media es la relación entre el cambio en la velocidad y el tiempo transcurrido.

amedia = Δv / Δt

Donde: amedia = Aceleración media, Δv = Cambio en la velocidad, Δt = Intervalo de tiempo
