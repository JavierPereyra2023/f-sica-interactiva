Velocidad Media

La velocidad media es la relación entre el desplazamiento total y el tiempo total transcurrido. Es un vector que indica qué tan rápido y en qué dirección se ha movido un objeto.

vmedia = Δx / Δt

Donde:

  • vmedia = Velocidad media
  • Δx = Desplazamiento
  • Δt = Intervalo de tiempo
